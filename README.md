# CAPM
This repository is used for verifying the maxpool-based CNN.

## Requirements

* python version 3.8.5
* packages:
  * numpy==1.19.2
  * torch == 1.8.0
  * torchvision == 0.9.0
  * psutil == 5.7.2
  * pandas == 1.1.3
  * onnx == 1.8.1
  * onnxruntime == 1.7.0
  
* These packages can be automatically installed by typing: 
  ```
  pip3 install requirements.txt
  ```
